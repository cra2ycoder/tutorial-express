# tutorial-express

## installation

```s
npm install express --save
npm install @types/express --save-dev
```

or in the `package.json` file, add the following config and hit `yarn install`

```js
"dependencies": {
    "express": "4.17.1"
}
"dependencies": {
    "@types/express": "4.17.0"
}
```
